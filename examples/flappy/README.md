FLAPPY
======

Flappy bird, the video game.

Avoid hitting pipes and rack up dozens of points.

![Screenshot](https://raw.githubusercontent.com/explodes/go-wo/master/examples/flappy/flappy.png)

# How to play

 - `Left/Right` to move
 - `Up` to flap around

# Reference highlights

 - Collision detection
 - Physics
 - Multi-scene game

# Running

`make run`
